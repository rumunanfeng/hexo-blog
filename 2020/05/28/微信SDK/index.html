微信SDK

发表于 2020-05-28 分类于 微信

微信的文档真是无力吐槽

最近需要在H5页面开发一个分享自定义链接给朋友的需求
所以就需要引入微信SDK
下面记录一下如何使用

使用SDK可分为两种：

  1. 一种是无需调用config，也无需公众号的操作，如从H5（在小程序的webview中）跳转回小程序
  2. 一种是需要调用config接口的操作，如自定义分享。这种同时还需要公众号
    的支持

1.引入sdk

这种方法非常简单，只需引入微信sdk再调用相应方法即可

引入方法有以下两种：

script引入

npm安装

再引入sdk

如果是无需调用config的方法，则到这步可以了。直接调用即可

2.公众号绑定域名

需要前往公众号将此项目域名绑定至JS接口安全域名，登录后可以在开发者中心看到拥有的接口权限

3.通过config接口注入权限验证配置

注意如果是IOS系统，则需要在第一个进入的页面就调用config接口。且全局也仅需调用一次。
安卓的话则需要每个使用到的页面都要调用

config接口中需要的参数是由后端接口提供，一般来说我们传给后端==当前页面完整地址==，后端再将我们需要的信息返回。

因为#号键对于url会有影响，所以最好是使用history模式。

参数jsApiList中必须要填写需要使用的JS接口列表，然后这个方法才可以用，例如我需要用到onMenuShareAppMessage方法。

4.通过ready接口处理成功验证

如此的话，在右上角点击三个点，再点击分享即可成功自定义分享了

参考微信SDK开发文档
